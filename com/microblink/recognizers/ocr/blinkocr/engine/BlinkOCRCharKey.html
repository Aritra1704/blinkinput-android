    Class representing a char in specific font. For example to define char 'k' in font Arial, use BlinkOCRCharKey keyK = new BlinkOCRCharKey((int)'k', BlinkOCRFont.ARIAL); For example to define char 'ü' in any font, use BlinkOCRCharKey keyU = new BlinkOCRCharKey((int)'ü', BlinkOCRFont.ANY); Created by dodo on 22/05/15.
