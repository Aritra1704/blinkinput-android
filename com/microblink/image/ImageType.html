    • Enum Constant Detail

      • ORIGINAL

        public static final ImageType ORIGINAL
        This is the original image that arrived from camera. If scanning region is set, you should also check getRoi method of the Image object because usually original images are in YUV_NV21 format.

      • DEWARPED

        public static final ImageType DEWARPED
        This is the dewarped image that is obtained after performing perspective correction of the item that will be scanned. This image is usually in format ALPHA_8, but might also be in BGRA_8888. This image is not always available - this depends on build settings and enabled recognizers.

      • GRAYSCALE

        public static final ImageType GRAYSCALE
        This is the grayscale version of the image that will be processed. Image will be in format ALPHA_8. This image is not always available - this depends on build settings and enabled recognizers.

      • COLOR_DROP

        public static final ImageType COLOR_DROP
        This is the grayscale version of the image that will be processed, where converting color image to gray levels was performed using color drop mechanism. Image will be in format ALPHA_8. This image is not always available - this depends on build settings and enabled recognizers.

      • SUCCESSFUL_SCAN

        public static final ImageType SUCCESSFUL_SCAN
        This is the original image that arrived from camera that was used to recognize item. If scanning region is set, you should also check getRoi method of the Image object because usually original images are in YUV_NV21 format.

      • OCR_PREPROCESSED

        public static final ImageType OCR_PREPROCESSED
        This is grayscale image preprocessed with OCR engine. This is the image that will be used for segmentation and classification of characters.

      • BEST_QUALITY

        public static final ImageType BEST_QUALITY
        This is the original image that arrived from camera that was selected as best possible from local set of camera frames. Usually this is the image that you will want to process. This image is not always available - this depends on build settings and enabled recognizers. If image is available, it will usually be in YUV_NV21 format.

      • DETECTION

        public static final ImageType DETECTION
        This is the image that was used as scratch surface to display document detection process (if any).
