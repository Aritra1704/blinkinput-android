    • Enum Constant Detail

      • CAMERA_DEFAULT

        public static final CameraType CAMERA_DEFAULT
        Will use device's default camera. If device has got only back facing camera, back facing camera will be used. If device has got only front facing camera, front facing camera will be used. If device has got both cameras, back facing camera will be used. NOTE: This option is the only possible option on Android 2.2 and earlier.

      • CAMERA_BACKFACE

        public static final CameraType CAMERA_BACKFACE
        Will use device's back facing camera, if it exists. If back facing camera does not exist, error will be raised. NOTE: This feature is available only for Android 2.3 and newer.

      • CAMERA_FRONTFACE

        public static final CameraType CAMERA_FRONTFACE
        Will use device's frontfacing camera, if it exists. If front facing camera does not exist, error will be raised.
